# My Portfolio
A portfolio that uses a old terminal-style UI to navigate thing about me and play tiny browser games.

### Quick Overview
* Fully static website
* Main files:
  * `index.html` — basic outline and container
  * `styles.css` — theme and styles
  * `script.js` — logic and games

### Features (Commands)

You type commands into the terminal and press enter to do things.
* help — opens the help window (this just lists commands).
* about/about me — opens the about me window.
* projects — opens a window with my projects and links to them.
* skills — opens window with my skills.
* contact — opens a window with my contacts.
* quote — gives some of my wisdom.
* surprise — ???.
* neo — matrix japanese falling text.
* color [] — changes the color scheme (green, purple, blue, orange, pink).
* theme [] — changes the theme (default, amber, monochrome, ms dos).
* snake — starts the Snake game.
* pong — starts the Pong game.
* clear — clears all the terminal's output.
